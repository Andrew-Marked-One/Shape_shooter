### Project #2
Timeline: 2023.12.3 - 2024.1.11

My first SFML and C++ project. I started game programming course first, but when time came to make first game, I stragled a lot with C++ and SFML

### Controls
- W: Move up
- S: Move down
- A: Move left
- D: Move right
- P: Pause
